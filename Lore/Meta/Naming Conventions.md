# Naming Conventions

*How things are named in this project and why.*

---

## Real Country Names vs. Lettered Codes

- **Real country name** (e.g. China, Japan, Nigeria) = refers to the actual present-day (2026) country and its current trajectory
- **Lettered code** (e.g. Country B, Country F) = refers to the fictional 2100s successor state extrapolated from that starting point
- The two are related but distinct — using a real name when discussing the fictional future state is an error
